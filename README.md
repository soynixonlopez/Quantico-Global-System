# Quantico System Global - Landing Page

Sitio web de ventas para **Quantico System Global**, distribuidor de cámaras de seguridad en Panamá.

## Tecnologías

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Variables de entorno y seguridad

1. Copia `.env.example` a `.env.local`.
2. Completa tus credenciales SMTP reales para el formulario de contacto.
3. Nunca subas `.env.local` ni contraseñas al repositorio.

El proyecto ya ignora archivos `.env*` locales vía `.gitignore`.

## Secciones del sitio

- Hero con CTA a WhatsApp
- Beneficios (imágenes de ejemplo)
- Marcas que distribuimos
- Catálogo de productos
- Estadísticas
- Por qué elegirnos
- Testimonios
- Contacto (formulario + email/WhatsApp)
